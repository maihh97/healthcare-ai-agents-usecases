# NHS AI Agent Network — Static Site

Live site can be accessed ad: https://maihh97.github.io/healthcare-ai-agents-usecases/


## Build Information

- **Framework**: Next.js 14 (App Router) with static export
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animation**: Framer Motion
- **Export command**: `next build` (configured for static export in `next.config.mjs`)
